# Генерация списка случайных чисел

## Описание задачи  
Программа оформлена в виде функции, генерирует список случайных чисел в диапазоне от 5 до 1100. Число элементов в списке составляет 21.  

## Среда и язык реализации  
- Язык: Python 3.12
- Среда разработки: Visual Studio  

## Используемые библиотеки  
- random – генерация случайных чисел
